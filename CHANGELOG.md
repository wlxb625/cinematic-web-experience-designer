# Changelog

## 1.1 — Scroll Choreography Upgrade

- Promoted scroll from a secondary animation technique to a mandatory master timeline when the brief requires a cinematic downward journey.
- Added explicit Trigger, Start, End, Pin, Scrub, normalized progress, mobile distance, and reduced-motion fields.
- Added scene continuity contracts: receive, preserve, transform, and hand off.
- Added production rules for GSAP ScrollTrigger, video scrubbing, image sequences, cleanup, and smooth-scroll boundaries.
- Added mandatory validation at 0%, 25%, 50%, 75%, and 100% progress, plus desktop/mobile recordings and reverse-scroll tests.
- Expanded the Tongxing example into a continuous scroll narrative.
- Added a React + GSAP reference scene and CSS implementation.
